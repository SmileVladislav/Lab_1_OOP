   - **Дія:** $C = a \cdot B$, де $a$ — константа .
2. **C7 (Тип даних):** $5110 \mod 7 = 0$
   - **Тип елементів матриці:** `double` (у Python реалізовано як `float`).

- `matrix.py` — основний програмний код, що реалізує множення на скаляр та розрахунок специфічної суми елементів.
